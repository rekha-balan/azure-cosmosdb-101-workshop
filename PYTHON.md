## Spatial Queries

[GeoJSON](http://geojson.org) format

The loaded documents look like this:
```
  ...
  {
    "location": {
      "type": "Point",
      "coordinates": [
        -73.34415,
        41.137996
      ]
    },
    "postal_cd": "06880",
    "country_cd": "US",
    "city_name": "Westport",
    "state_abbrv": "CT",
    "latitude": 41.137996,
    "longitude": -73.34415
  },
  ...
```

Query:
```
SELECT * FROM root WHERE ST_DISTANCE(root.location, {'type': 'Point', 'coordinates': [-73.34415, 41.137996] }) < 1000
```
